# 等保三级 (MLPS 3.0) 核心自查表

> **适用范围**: 涉及关键信息基础设施或大量公民个人信息的系统。

## 1. 网络安全
- [ ] **入侵检测**: 部署 IDS/IPS 对关键节点进行监控。
- [ ] **访问控制**: 网络区域划分（DMZ、内网、数据库区），配置 ACL。

## 2. 主机安全
- [ ] **身份鉴别**: 双因素认证 (2FA) 是强制的。
- [ ] **访问控制**: 最小权限原则，禁止直接使用 root/admin 登录。
- [ ] **安全审计**: 必须开启操作日志，留存时间 >= 6个月。

## 3. 应用安全
- [ ] **通信完整性**: 全站 HTTPS (TLS 1.2+)。
- [ ] **数据保密性**: 敏感字段 (身份证、手机号) 加密存储 (AES-256)。
- [ ] **抗抵赖**: 关键操作必须记录 IP、时间、用户 ID。

## 4. 数据备份
- [ ] **异地备份**: 必须有异地数据备份机制。
- [ ] **恢复演练**: 每年至少 1 次灾难恢复演练。
