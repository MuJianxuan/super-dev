# 个人信息保护法 (PIPL) 合规清单

> **适用范围**: 所有在中国境内处理自然人个人信息的应用。

## 1. 最小必要原则
- [ ] **收集范围**: 仅收集业务功能所必需的信息。(例如：外卖 App 不应收集通讯录)。
- [ ] **权限申请**: 不得因用户拒绝非必要权限而拒绝提供基本服务。

## 2. 告知同意
- [ ] **隐私政策**: 首次启动必须弹窗展示，并需用户主动勾选（默认勾选违规）。
- [ ] **单独同意**: 涉及敏感个人信息（人脸、位置、金融账号）需取得单独同意。

## 3. 用户权利
- [ ] **查阅复制**: 用户必须能查询和复制自己的个人信息。
- [ ] **更正删除**: 提供更正和注销账号的入口（注销难是重灾区）。
- [ ] **撤回同意**: 必须提供撤销已授权权限的开关。

## 4. SDK 管理
- [ ] **披露**: 隐私政策中必须列出所有第三方 SDK 及其用途。
- [ ] **初始化**: 用户同意隐私政策前，严禁初始化任何 SDK。
