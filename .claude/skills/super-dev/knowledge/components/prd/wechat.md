## {{ section_number }}. 小程序特定需求 (Mini-Program)

### {{ section_number }}.1 生态限制
- **包体积**: 主包 < 2MB。功能模块必须使用"分包" (Sub-packages)。
- **域名白名单**: 所有 API 必须为 HTTPS 且在后台配置白名单。
- **类目资质**: 功能必须符合营业执照范围（如：无"视频"资质不可做直播）。

### {{ section_number }}.2 微信集成
- **登录流程**: 静默 `wx.login` -> `code` -> 后端换取 `OpenID`。（不再支持一键获取头像昵称）。
- **手机号**: 需调用付费组件（实时验证），仅在必要高信任场景使用。
- **支付**: 原生微信支付集成。*注意：iOS 端严禁虚拟支付（知识付费/会员），否则会封禁。*

### {{ section_number }}.3 社交裂变
- **分享卡片**: 必须定义 `onShareAppMessage`，自定义 5:4 比例图片。
- **活动**: "邀请好友"等裂变逻辑需符合微信运营规范，避免被判定为诱导分享。
