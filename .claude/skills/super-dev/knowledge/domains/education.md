```python
# 知识点掌握度计算
def calculate_mastery(user_attempts):
    """
    计算用户对知识点的掌握度
    """
    if not user_attempts:
        return 0.0

    # 最近N次练习权重更高
    weights = [0.5 ** i for i in range(len(user_attempts))]
    weights.reverse()

    # 加权正确率
    weighted_score = sum(
        attempt.is_correct * weight
        for attempt, weight in zip(user_attempts, weights)
    ) / sum(weights)

    # 考虑练习间隔 (遗忘曲线)
    time_gap = (now - user_attempts[-1].timestamp).days
    forgetting_factor = 0.9 ** time_gap

    return weighted_score * forgetting_factor
```

## 10. 常见陷阱 (PITFALLS)

### 10.1 技术陷阱

- ❌ **忽视网络环境**: 视频卡顿、断线
- ❌ **兼容性问题**: 设备/浏览器差异
- ❌ **不处理离线**: 无网络无法学习
- ❌ **忽视并发**: 考试期间服务器崩溃

### 10.2 产品陷阱

- ❌ **过度游戏化**: 分散学习注意力
- ❌ **忽视学习效果**: 只看完成率不看学习效果
- ❌ **一刀切**: 不考虑不同年龄段需求
- ❌ **内容质量差**: 影响口碑和复购

### 10.3 运营陷阱

- ❌ **忽视教师体验**: 备课工具难用
- ❌ **忽视家长参与**: 家长无法了解进度
- ❌ **过度营销**: 承诺无法实现的提分效果
- ❌ **忽视退费**: 退费流程复杂
